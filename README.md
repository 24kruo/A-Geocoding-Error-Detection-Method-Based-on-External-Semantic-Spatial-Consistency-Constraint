# A-Geocoding-Error-Detection-Method-Based-on-External-Semantic-Spatial-Consistency-Constraint

Data in this repository is for the paper **A-Geocoding-Error-Detection-Method-Based-on-External-Semantic-Spatial-Consistency-Constraint**.

Below is an overview of each file in this repository.

  - `algorithm.py` an algorithm for error identification based on external semantic-spatial consistency constraint
  - `Data` 
    - `address data.xlsx` Test data for evaluating the algorithm’s performance
    - `reference POI.xlsx` External reference POI data used for constraints
